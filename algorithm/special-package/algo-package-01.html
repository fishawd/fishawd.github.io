# 01背包

简介

# 题目

有 N 件物品和一个容量为 V 的背包。第 i 件物品的体积是c[i]，价值是 w[i]。求解将哪些物品装入背包可使价值总和最大。 (每个物品的数量都是1，要么放入要么不放入，所以该问题叫做01背包)

# 解题思路

动态规划算法首先要找到状态转移方程。根据题目要求，我们可以知道每个物品有两个状态，放入或者不放入，这就找到了子问题。 我们用一个二维数组 dp[i][v] 表示前 i 个物品，背包容量为v的最大价值( i 的取值范围是 1~N，v的取值范围是 1~V )，

我们根据划分的子问题，第 i 个物品是否选择，得到第 i 个物品的状态。

如果第 i 个物品不选， 那么问题就转化为 第 i-1 个物品，背包容量为v时的最大价值，即 dp[i][v] = dp[i-1][v]。

如果第 i 个物品选了，体积为c[i]，那么此时剩余容量为 v-c[i]，那么问题就转化 第 i-1 个物品，背包容量为 v-c[i]的最大价值，那么第 i 个物品的最大价值就是 dp[i][v] = dp[i-1][v-c[i]] + w[i]

我们取这两种状态中最大值，于是可以得到状态转移方程：

dp[i][v] = max{dp[i-1][v], dp[i-1][v-c[i]] + w[i]}

# 代码实现

二维数组动态规划

class Solution {
    // 数组c中存放的是物品，c[i]表示体积
    // w中存放的是c中物品对应的体积
    // V是背包容量
    public int pack01(int[] c, int[] w, int V) {
        int[][] dp = new int[c.length + 1][V + 1];
        //这里应该初始化dp[0][v] = 0，表示一个物品都不放，背包最大价值为0， 由于创建时候默认都是0,不再手动初始化
        // 枚举物品
        for (int i = 1; i <= c.length; i++) {
            // 枚举背包容量
            for (int v = 0; v <= V; v++) {
                // 选了，当 背包容量大于等于c[i]时候才能往里放
                // 这里c[i-1] w[i-1]是因为c w的下标从0开始
                if (v >= c[i - 1]) {
                    dp[i][v] = Math.max(dp[i - 1][v], dp[i - 1][v - c[i - 1]] + w[i - 1]);
                } else {
                    // 不选
                    dp[i][v] = dp[i - 1][v];
                }
            }
        }
        return dp[c.length][V];
    }
}

# 优化思路

从上边的解法中可以看出，该方法的时间复杂度和空间复杂度均为 O(NV)。仔细观察发现，状态转移方程的第一维都是一样的。换句话说，就是当前状态只与上一个状态的第二维有关。因此我们考虑去掉第一维的状态，那么状态转移方程就变为：

dp[v] = Math.max(dp[v], dp[v - c[i]] + w[i])

那么问题就来了，在二维的时候， dp[v - c[i]] 的一维部分是dp[i-1]，也就是说 dp[v - c[i]] 使用的是 i-1 的状态， 而直接去掉一维后，由于 v-c[i] < v，v从0到N， 那么 dp[v-c[i]] 是先于 dp[v] 计算的， 也就是说计算 dp[v] 的时候使用的是 i 的状态，显然是错误的。

解决办法就是将 V 倒序遍历。 对于每个物品 i， 都会产生一个 dp 数组。这句话的意思是说， 当 i=2 时， dp 数组一定是由 i=1 产生的，也就是说，当遍历到 i 时， dp 数组中保存的是 i-1 的状态。 由于 v-c[i] < v， 当倒序遍历的时候， dp[v] 依赖的状态 dp[v - c[i]] 在本轮并没有计算，dp[v - c[i]]仍然保存的是 i-1 的状态。这个时候，状态转移方程使用的就是 i-1 的状态了。

# 优化代码

一维数组动态规划

class Solution {
    public int pack01(int[] c, int[] w, int V) {
        int[] dp = new int[V + 1];
        // 应该初始化dp[v]=0,表示初始状态为0
        for (int i = 1; i <= c.length; i++) {
            for (int v = V; v >=0 ; v--) {
                if (v >= c[i - 1]) {
                    dp[v] = Math.max(dp[v], dp[v - c[i - 1]] + w[i - 1]);
                }
            }
        }
        return dp[V];
    }
}
