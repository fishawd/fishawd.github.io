# 栈的定义

栈是一种受限的线性表，只允许在栈的末尾进行插入或删除操作。栈有两种实现方式，一种用数组实现，叫顺序栈；另一种用链表实现，叫链栈。表头叫栈底，表尾叫栈顶。插入新元素叫进栈，删除元素出栈。

栈的示意图

# 栈的特性

  • 后进先出(last-in,first-out, 缩写为LIFO)。即后进栈的元素先出栈。

# 顺序栈

  • 进栈和出栈

顺序栈就是用数组模拟后进先出。在下面我们定义了一个容量为5的顺序栈，同时定义了一个始终指向栈顶的指针top。top = -1表示栈空，top = length - 1表示栈满。

public class SequeStack {
    // 定义顺序栈的容量大小
    public int[] data = new int[5];
    // 定义栈顶指针的位置，top = -1表示为空, top = data.length - 1 栈满
    public int top = -1;

    // 进栈
    public void push(int value){
        // 栈不满
        if (top < data.length - 1){
            data[++top] = value;
        }
        // 栈满可以进行扩容操作， 这里就不再写了
    }
    // 出栈
    public int pop(){
        // 栈不空
        if (top > -1){
            return data[top--];
        }
        return -1;
    }
    public static void main(String[] args) {
    SequeStack sequeStack = new SequeStack();
    sequeStack.push(2);
    sequeStack.push(3);
    sequeStack.push(4);
    sequeStack.push(1);

    int value = sequeStack.pop();
    while (value != -1){
        System.out.println(value);
        value = sequeStack.pop();
        }
    }
}

# 链栈

  • 进栈和出栈

链栈的结点定义和单链表一样。进栈就是单链表的头插法，出栈就是删除单链表的头结点。这里和链表一样定义一个fist不存储数据，当 first.next = null 时栈为空。因为是链表结构，所以也就不存在栈满的情况。

class LinkStack {
    Node first = new Node(-1, null);

    // 进栈，也就是单链表的头插法
    public void push(int value) {
        Node newNode = new Node(value);
        newNode.next = first.next;
        first.next = newNode;
    }

    // 出栈，也就是删除单链表的头结点
    public int pop(){
        int res = -1;
        if (first.next != null){
            Node head = first.next;
            first.next = head.next;
            res = head.value;
        }
        return res;
    }
}
