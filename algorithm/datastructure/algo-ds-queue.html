# 队列

简介

本文介绍了队列的定义、特性和实现，包括顺序队列，循环队列和链队。

# 队列的定义

队列和栈一样，也是一种受限的线性表。队列也包括顺序队和链队。队列只允许在队头进行删除，也就是出队操作，在队尾插入，也就是进队操作。

队列的示意图

# 队列的特性

  • 先进先出(first-in, first-out，简称FIFO)

# 队列的实现

队列的实现包括顺序队和链队，其中比较特殊的一种顺序队列叫循环队列。

# 顺序队

  • 进队和出队

在顺序队中，我们定义了队头指针 front 和队尾指针 rear，初始化为0。进队时候先赋值 data[rear] = value，然后 rear 自增1， 出队时候先弹出元素，然后 front 自增1。 判断队空的条件是 front = rear。

顺序队列这样会导致一个问题：随着元素的入队和出队，队尾指针最终会指向数组最末位置，而队头指针会指向中间位置，此时虽然 rear = length，但是其实队列中的空间并没有用完，但也无法入队，因此被称为假溢出。处理假溢出的方法有多种，一种是判断假溢出了就把元素整体向前移动，另一种是每出队一个元素就把元素整体向前移动，使得队头指针始终指向数组最初位置0，还有一种是使用循环队列。此外，也可以使用链队。

在顺序队中，判断假溢出的条件是 rear == length && front != 0， 即队尾指针移动到了数组最末位置，队头指针不为0.

public class SequeQueue {
    // 创建容量为5的队列
    int[] data = new int[5];
    // 队头指针
    int front = 0;
    // 队尾指针
    int rear = 0;

    // 进队，
    public void enqueue(int value) {
        // 判断是否满了
        if (front == 0 && rear == data.length) {
            // 这里可以做一些扩容操作，就不再演示了
            return;
        }
        // 这里队尾指针指向最后，但是front不为0,即为假溢出
        if (rear == data.length) {
            // 这里可以做将元素整体前移再进队的操作
        } else {
            // 进队
            data[rear++] = value;
        }
    }

    // 出队， 队列为空的条件是 front = rear
    public int dequeue() {
        if (front == rear) {
            return -1;
        }
        return data[front++];
    }

    public static void main(String[] args) {
        SequeQueue sequeQueue = new SequeQueue();
        sequeQueue.enqueue(2);
        sequeQueue.enqueue(4);
        sequeQueue.enqueue(3);
        sequeQueue.enqueue(1);
        int value = sequeQueue.dequeue();
        while (value != -1) {
            System.out.println(value);
            value = sequeQueue.dequeue();
        }
    }
}

  • 循环队列

所谓循环队列，就是队头队尾首位相接，主要是为了充分利用空间，解决假溢出的问题。 如图所示，所谓的循环队列并不是真正的内存地址是首尾相连的，而是在逻辑上是一个循环，本质上还是一个数组。 我们同样定义队头指针 front 和 队尾指针 rear。 假设队列容量为n，当队空时候有 front = rear, 当队满时候也有 front = rear，因此我们规定循环队列最多只能存储 n-1 个元素。那么队空时候就是 front = rear，队满的条件就是 front = (rear + 1) % n。

循环队列示意图

class CircularQueue {
    int n = 5;
    int[] data = new int[n];
    int front = 0;
    int rear = 0;

    // 进队
    public void enqueue(int value) {
        // 如果队不满
        if ((rear + 1) % n != front) {
            data[rear] = value;
            // 队尾指针向后移动
            rear = (rear + 1) % n;
        }
    }

    // 出队
    public int dequeue() {
        int res = -1;
        //如果不为空
        if (rear != front) {
            res = data[front];
            // 队头指针移动
            front = (front + 1) % n;
        }
        return res;
    }

    // 返回队列中元素个数
    public int size() {
        return (rear - front + n) % n;
    }
}

# 链队

  • 进队和出队

链队的实现也是用链表来写，结点定义也一样。那么为了实现先进先出，显然我们需要一个头指针和一个尾指针，进队就是尾插法，出队就是删除头结点。头指针 front.next 指向第一个元素， 而尾指针 rear 保存最后一个元素。 队列为空的条件就是 front.next = null，链队无须判断是否队满。

class LinkQueue{
    Node front = new Node();
    Node rear = front;

    // 进队
    public void enqueue(int value){
        Node node = new Node(value);
        rear.next = node;
        rear = node;
    }

    // 出队
    public int dequeue(){
        int res = -1;
        // 队不空
        if (front.next != null){
            Node node = front.next;
            front.next = node.next;
            res = node.value;
            // 如果出队的是最后一个元素，重新修改把front赋值给rear
            if (front.next == null){
                rear = front;
            }
        }
        return res;
    }
}
