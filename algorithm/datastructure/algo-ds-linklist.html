# 链表的定义

链表是一种基础的线性数据结构。和数组不同，数组中的线性数序是由数组下标决定的，而链表中的顺序是由各个对象中的指针决定的。各个结点在物理地址上不连续，但在逻辑上是连续的。

单链表存储示意图

链表（Linked list）是一种常见的基础数据结构，是一种线性表，但是并不会按线性的顺序存储数据，而是在每一个节点里存到下一个节点的指针(Pointer)。 ---来自维基百科

# 链表的特性

  • 在表头表尾插入和删除元素很快，时间复杂度可以达到O(1)。
  • 没有容量限制。
  • 随机查找元素和随机删除元素比较慢，因为需要遍历。

# 插入
  1. 头插法

头插法就是每次将结点插在头结点前面，使得新插入的结点变为新的头结点。一般情况下，为了方便各种操作，我们都会设置一个哑结点 dummy 作为头结点，哑结点不存储数据，只指向第一个存储数据的结点。

class LinkList{
    Node dummy = new Node(-1, null);
    // 头插法
    public void headInsert(Node node){
       node.next = dummy.next;
       dummy.next = node;
    }
}

  1. 尾插法

尾插法就是将结点插入最后一个节点后面，使之成为新的尾结点。

class LinkList{
    Node dummy = new Node(-1, null);
    Node tail;
    // 尾插法，需要一个变量保存尾结点
    public void tailInsert(Node node){
        if (tail == null){
            dummy.next = node;
        }else {
            tail.next = node;
        }
        tail = node;
        tail.next = null;
    }
}

public void removeNode(int value) {
    Node cur = dummy;

        }
        cur = cur.next;
    }
    cur.next = cur.next.next;
}

# 双链表

顾名思义，双链表就是每个结点除了保存指向后继的指针外，还保存了指向前驱的指针。

# 插入
  1. 头插法
class DoubleLinkList {
    DoubleNode dummy = new DoubleNode(-1, null, null);
    // 头插法
    public void headInsertDouble(DoubleNode node) {
        DoubleNode head = dummy.next;
        // 如果链表为空
        if (head == null){
           dummy.next = node;
           node.pre = dummy;
           node.next = null;
        }else {
            node.next = head;
            head.pre = node;
            dummy.next = node;
        }
    }
}

# 经典例题

  1. 输入一个链表，输出该链表中倒数第k个结点。

这道题目是经典的快慢指针问题，我们只需要先让快指针先走到第k个结点，然后慢指针再出发，等到快指针走到最后，慢指针就指向倒数第k个节点了。

    public Node find(int k) {
        Node fast = dummy.next;
        Node slow = fast;
        int i = 0;
        while (fast != null) {
            i++;
            fast = fast.next;
            if (i > k) {
                slow = slow.next;
            }
        }
        return slow;
    }

    public void reverse() {
        Node cur = dummy.next;
        dummy.next = null;
        Node temp;
        while (cur != null) {
            temp = cur;
            cur = cur.next;
            temp.next = dummy.next;
            dummy.next = temp;
        }
    }

解法3：递归

    public Node reverse3(Node head){
        // 递归终止条件
        if (head == null || head.next == null){
            return head;
        }else {
            Node newHead = reverse3(head.next);
            // 改变结点指向，也就是后一个节点指向前一个结点
            head.next.next = head;
            head.next = null;
            return newHead;
        }
    }
