# 二叉树的定义

树是由n个结点组成的一种非线性的数据结构，当 n=0 时为空树。二叉树是树的一种特殊情形，每个结点最多有两个分支，因此叫二叉树。

二叉树示意图

# 基本概念

树根：最开始的一个结点，有且仅有一个，在最上层，叫树根(root)。如上图中的A结点就是树根。

子树：每一个非空节点拥有的树。上图中 B、D、E 组成的树是A的左子树，C、F组成的树是A的右子树。

父结点：某个结点的父亲。上图中A结点称为B和C的父结点。

孩子结点：某个结点的孩子。 上图中B和C称为A结点的孩子节点。

兄弟结点：拥有同一父节点的结点。上图中B和C称为兄弟结点。

堂兄弟结点：某结点的父节点在同一层次，或者说祖父结点相同。上图中D和F，E和F就是堂兄弟。

结点的度：某个结点拥有的子树的数量。在二叉树中，结点的度只能为0，1，2。

树的度：树中结点的度的最大值。

叶子结点：度为0的结点，也就是没有子树。上图中D、E、F为叶子结点。

层次：从根结点开始，根节点在第一层，根节点的孩子在第二层，以此类推。

深度(高度)：树中结点的最大层次。

# 二叉树的类型

完全二叉树：设二叉树的高度为h，除第 h 层外，其它各层 (1～h-1) 的结点数都达到最大个数，第h层有叶子结点，并且叶子结点都是从左到右依次排列，叫做完全二叉树。下图就是一棵完全二叉树。

满二叉树：高度为h的二叉树中结点数目达到最大值，即2h-1，如下图所示。除叶子结点外，其他各个结点的度都为2。满二叉树是完全二叉树的一种特例。

二叉排序树：按照一定顺序排列的二叉树。规则是 所有的左孩子小于父结点的值，所有的右孩子大于父结点的值。同时，每个结点的左子树的所有结点的值都小于该节点，每个结点的右子树的所有结点的值都大于该节点。二叉排序树的中序遍历是一个升序的序列。

平衡二叉树：平衡二叉树是一种特殊的二叉排序树。它的特点是：一棵空树或者它的结点的左右两个子树的高度差的绝对值不超过1，并且左右两个子树都是一棵平衡二叉树。

# 二叉树的特性

  • 二叉树的第h层上至多有 2h-1 个结点，h >= 1 (下同)。
  • 深度为h的二叉树至多有 2h-1 个结点。
  • 在一棵非空二叉树中， 结点数 = 总的度数 + 1。

证明：假设结点数为 n, 度为1的结点个数有 n1个，度为2的结点个数有n2个，那么n个结点就有 n-1 条边， n-1 = n1 + 2n2， 即 n = n1 + 2n2 + 1 。

  • 如果一棵非空二叉树的度为0的结点(叶子结点)个数为 n0，度为2的结点数为n2，那么 n0 = n2 + 1。

证明：假设结点数为n，度为0的结点个数有 n0个， 度为1的结点个数有 n1个，度为2的结点个数有n2个, 那么 n = n0 + n1 + n2。 从上一条性质可知 n = n1 + 2n2 + 1, 那么 n0 + n1 + n2 = n1 + 2n2 + 1，即n0 = n2 + 1。

  • 具有n个结点的完全二叉树的深度为 [log2n]+1 (向下取整)

# 二叉树的结点定义

从上边的结构图中我们可以知道每个结点应该有三个域，一个数据域，两个指针域，分别指向左右两个孩子。

class TreeNode {
    // 元素值
    int value;
    // 左孩子
    TreeNode left;
    // 右孩子
    TreeNode right;
    
    TreeNode(){}
    TreeNode(int value){
        this.value = value;
    }
    TreeNode(TreeNode left, TreeNode right, int value){
        this.left = left;
        this.right = right;
        this.value = value;
    }
}

# 先序遍历

先序遍历的规则是 先访问根结点，然后访问左子树，最后访问右子树。

# 先序遍历的递归实现

class BinaryTree {

    // 先序遍历递归实现
    public void preorder(TreeNode root) {
        if (root == null) {
            return;
        }
        System.out.println(root.value);
        inorder(root.left);
        inorder(root.right);
    }
}

# 先序遍历的非递归实现

非递归遍历使用了一个辅助栈保存结点。该方法的流程是首先访问当前结点和其所有的左孩子结点，依次将访问过的节点入栈，直到找到最左边的结点。然后出栈，将当前结点赋值为最左边结点的右孩子结点。重复这个过程。

class BinaryTree {

    // 先序遍历非递归实现
    public void preorderNotRecursion(TreeNode root) {
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        while (!stack.isEmpty() || cur != null) {
            //先访问结点，再把结点入栈
            while (cur != null) {
                System.out.println(cur.value);
                stack.push(cur);
                cur = cur.left;
            }
            if (!stack.isEmpty()) {
                cur = stack.pop();
                cur = cur.right;
            }
        }
    }
}

# 中序遍历

中序遍历的规则是 先访问左子树，再访问根节点，最后访问右子树。二叉排序树的中序遍历的是一个有序的序列。

# 中序遍历的递归实现

class BinaryTree {
    // 中序遍历递归实现
    public void inorder(TreeNode root) {
        if (root == null) {
            return;
        }
        inorder(root.left);
        System.out.println(root.value);
        inorder(root.right);
    }
}

下面的中序遍历的非递归实现同样需要一个辅助栈。

class BinaryTree {

    public void inorderNotRecursion(TreeNode root) {
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        while (!stack.isEmpty() || cur != null) {
            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }
            if (!stack.isEmpty()) {
                cur = stack.pop();
                System.out.println(cur.value);
                cur = cur.right;
            }
        }
    }
}

# 后序遍历

中序遍历的规则是， 先访问左子树，再访问右子树，最后访问根节点。

# 后续遍历的递归实现

class BinaryTree {
    // 后续遍历递归实现
    public void postorder(TreeNode root) {
        if (root == null) {
            return;
        }
        postorder(root.left);
        postorder(root.right);
        System.out.println(root.value);
    }
}

# 后续遍历的非递归实现

下面的后续遍历的非递归实现使用了两个栈。

class BinaryTree {
    // 后序遍历非递归实现
    // 双栈法
    public void postorderNotRecursion(TreeNode root) {
        Stack<TreeNode> stack1 = new Stack<>();
        Stack<TreeNode> stack2 = new Stack<>();
        TreeNode cur = root;
        if (cur != null) {
            stack1.push(cur);
        }
        while (!stack1.isEmpty()) {
            cur = stack1.pop();
            stack2.push(cur);
            if (cur.left != null) {
                stack1.push(cur.left);
            }
            if (cur.right != null) {
                stack1.push(cur.right);
            }
        }
        while (!stack2.isEmpty()) {
            System.out.println(stack2.pop().value);
        }
    }
}

# 层次遍历

顾名思义，层次遍历就是按照二叉树的层次，从上到下，从左到右，一层一层进行遍历。层次遍历需要使用队列进行辅助操作，因为队列先进先出的特性刚好符合层次遍历的需要。

class BinaryTree{
    // 层次遍历
    public void levelorder(TreeNode root) {
        Queue<TreeNode> queue = new LinkedList<>();
        TreeNode cur = root;
        if (cur != null) {
            queue.add(cur);
        }
        while (!queue.isEmpty()){
            cur = queue.poll();
            System.out.println(cur.value);
            if (cur.left!=null){
                queue.add(cur.left);
            }
            if (cur.right !=null){
                queue.add(cur.right);
            }
        }
    }
}

# 插入结点

class BinaryTree {
    // 二叉排序树插入结点
    public TreeNode createBST(TreeNode root, int value) {
        if (root == null) {
            root = new TreeNode(null, null, value);
        } else if (value <= root.value) {
            root.left = createBST(root.left, value);
        } else {
            root.right = createBST(root.right, value);
        }
        return root;
    }

    public static void main(String[] args) {
        BinaryTree binaryTree = new BinaryTree();
        TreeNode root = new TreeNode(null, null, 3);
        binaryTree.createBST(root, 1);
        binaryTree.createBST(root, 2);
        binaryTree.createBST(root, 5);
        binaryTree.createBST(root, 4);
        binaryTree.createBST(root, 6);
        binaryTree.createBST(root, 7);
        // 3125467
        binaryTree.preorder(root);
        // 1234567
        binaryTree.inorder(root);
    }
}

# 计算二叉树的深度

计算二叉树的深度只需要根据树的遍历方法添加返回值即可，下面给出一种DFS递归解法，当然也可以使用层次遍历，找出最大层。

// 计算二叉树的深度
public int deepth(TreeNode root) {
    if (root == null) {
        return 0;
    }
    int left = deepth(root.left);
    int right = deepth(root.right);
    return Math.max(left, right) + 1;
}
